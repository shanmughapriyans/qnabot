# 🤖 QnABot

A smart Question & Answer Bot powered by AI.

## Overview

QnABot is an intelligent conversational assistant designed to answer questions accurately and efficiently using modern AI techniques.

## Features

- 💬 Natural language question answering
- 🔍 Context-aware responses
- ⚡ Fast and efficient query processing
- 🧠 Powered by large language models

## Getting Started

### Prerequisites

- Python 3.9+
- pip

### Installation

```bash
git clone https://github.com/shanmughapriyans/qnabot.git
cd qnabot
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
